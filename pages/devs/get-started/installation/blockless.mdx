# Blockless CLI

Blockless CLI will need to be installed in order to invoke [Inference Generation](./topic-creators/create-deploy-inference-generation-function) and [Weight Adjustment](./topic-creators/create-deploy-weight-adjustment-function) functions when in the process creating a topic on Allora.

## Installation

- Access the [Blockless Quick Start Guide](https://blockless.network/docs/network/quick-start).
- Follow the instructions to download and install the Blockless CLI on your system.
- Create your first sample function using the guide.
