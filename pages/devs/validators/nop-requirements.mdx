# System Requirements

You can use any modern Linux distribution to run an Allora validator.  
Internally we use **Debian 12** x86_64.

## MAINNET and TESTNET validators' requirements:

CPU - 4/8 Core  
Memory - 8/16GB  
Disk - SSD or NVMe  
Size - 512GB or larger

## EDGENET validators' requirements:

CPU - 1/2 Core  
Memory - 2/4 GB  
Disk - SSD or NVMe  
Size - 1GB or larger
