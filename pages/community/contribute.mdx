# Contributions

## Developers

[Our open-source repos](https://github.com/allora-network) all follow the [same contribution guidelines](https://github.com/allora-network/allora-chain/blob/main/CONTRIBUTING.md), including these docs themselves.

Our [Discord](https://discord.gg/allora) is great for speaking directly with the core Allora developers, coordinating with fellow [Developer Advocates](https://www.allora.network/blog/introducing-the-allora-network-community-advocate-program-4258c), and keeping up to date with the latest regarding all things Allora.

Follow [our Twitter](https://twitter.com/AlloraNetwork) to stay aware of new announcements and releases.

## Non-Technical Contributions

Head over to our [Discord](https://discord.gg/allora) to learn more about the many ways you can contribute to Allora.
